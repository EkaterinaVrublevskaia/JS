7. *Сравнить null и 0. Попробуйте объяснить результат. Null - используется чтобы указать на отсутсвие вообще какого-либо значения, 0 же это уже конкретное значение.
Но сравним Null c 0 и посмотрим, что на результаты:
alert( null > 0 ); // false
alert( null == 0 ); // false
alert( null >= 0 ); // true
Сравнения преобразуют null в число, рассматривая его как 0 => выражение null >= 0 истинно, а null > 0 ложно.
С другой стороны, для нестрогого равенства '==' значений null действует особое правило: эти значения ни
к чему не приводятся, они равны друг другу и не равны ничему другому => null == 0 ложно.
